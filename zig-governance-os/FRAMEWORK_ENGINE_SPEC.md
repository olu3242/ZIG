# Framework Engine Spec

The Framework Intelligence Engine supports ISO 27001:2022, NIST CSF 2.0, NIST 800-53 Rev 5, SOC 2, HIPAA, PCI DSS, CIS Controls v8, GDPR, CMMC, and custom frameworks.

Core registry entities: `frameworks`, `framework_versions`, `framework_domains`, `framework_controls`, `framework_requirements`, `framework_mappings`, and `framework_crosswalks`.

Dashboard outputs include framework coverage, readiness, health, control coverage, evidence coverage, and gap analysis.
