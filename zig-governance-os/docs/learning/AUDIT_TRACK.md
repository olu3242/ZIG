# Audit Track

Trains learners on audit planning, evidence sampling, and findings — backed by
`AuditService` (`audits`, audit trail via `auditEvents`).

## Learning path: Audit Planning and Execution

| Module | Type | Duration |
|---|---|---|
| Scoping an Audit Against a Framework | lesson | 25 min |
| Evidence Sampling Methods | lesson | 30 min |
| Writing Defensible Audit Findings | lesson | 30 min |
| Plan an ISO 27001 Internal Audit for HealthBridge | lab | 55 min |
| Draft Findings and Corrective Actions for HealthBridge | exercise | 45 min |

## Notes

- Maps to `AuditService.recordAction`/`Audit` records; reinforces evidence-to-control
  traceability rather than free-floating findings.
- Final exercise uses the HealthBridge scenario (`docs/scenarios/HEALTHBRIDGE.md`).
