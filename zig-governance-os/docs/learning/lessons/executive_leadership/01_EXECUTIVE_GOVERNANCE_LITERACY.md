# 01_EXECUTIVE_GOVERNANCE_LITERACY

## Objectives
- Read a governance score and its four explainable inputs without needing a translator.
- Identify what questions a board should ask about a governance score.
- Recognize when a governance score is improving for real vs. cosmetically.

## Business Context
Executives consume governance data secondhand by default — this lesson builds direct
literacy so leadership can challenge a report instead of just accepting it.

## Scenario Mapping
GovSec (`docs/scenarios/GOVSEC.md`) — read its unreported governance score directly.

## Framework Mapping
Trains on `GovernanceScore` fields directly (same data as Governance Track Lesson 05, from
an executive consumption angle).

- Real-vs-cosmetic score improvement comparison diagram

## Knowledge Check
1. What question should a board ask if `evidenceCoverage` is low but the score is rising?
2. What's the difference between a real and a cosmetic governance score improvement?
