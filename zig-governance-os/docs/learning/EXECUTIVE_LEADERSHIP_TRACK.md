# Executive Leadership Track

Trains learners on reading governance scores and communicating them to a board/executive
audience — backed by `GovernanceService` (`governance_scores`, `recommendations`).

## Learning path: Governance Communication for Executives

| Module | Type | Duration |
|---|---|---|
| What a Governance Score Does and Does Not Tell You | lesson | 25 min |
| Translating Recommendations into Business Risk Language | lesson | 30 min |
| Structuring an Executive Governance Report | lesson | 25 min |
| Build a Board-Ready Report for GovSec | lab | 55 min |
| Present GovSec's Governance Roadmap to a Mock Board | exercise | 40 min |

## Notes

- Maps to `GovernanceScore` records (explainable score fields: `controlsImplemented`,
  `evidenceCoverage`, `riskTreatment`, `assessmentCompletion`, `explanation`) and
  `GovernanceService.findRecommendations`.
- Final exercise uses the GovSec scenario (`docs/scenarios/GOVSEC.md`).
