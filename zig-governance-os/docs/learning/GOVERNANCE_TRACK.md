# Governance Track

Trains learners on the Universal Governance Model, governance scoring, and Mission
Control — backed by `GovernanceService` (`governance_scores`, `recommendations`).

## Learning path: Governance Program Fundamentals

| Module | Type | Duration |
|---|---|---|
| The Universal Governance Model: Organization to Report | lesson | 25 min |
| Designing a Governance Program from Scratch | lesson | 35 min |
| Reading and Explaining a Governance Score | lesson | 30 min |
| Build a Governance Program for GovSec | lab | 60 min |
| Present a Governance Score Improvement Plan for GovSec | exercise | 45 min |

## Notes

- Maps to `GovernanceService.findRecommendations` and `GovernanceScore` records
  (read-only training content; no new service methods required).
- Final exercise uses the GovSec scenario (`docs/scenarios/GOVSEC.md`).
