# Continuous Compliance Specification

Continuous compliance calculates the live posture across framework readiness, control health, evidence health, audit readiness, and certification readiness.

## Implementation

- Package: `packages/continuous-compliance`
- Engine: `ContinuousComplianceEngine`
- Output: compliance score and red, amber, green posture band
- Supported domains: ISO 27001, SOC 2, HIPAA, PCI DSS, NIST, CIS, custom frameworks
