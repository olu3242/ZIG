# Agent OS Specification

Zig Agent OS provides a typed autonomous workforce registry for compliance, risk, audit, policy, vendor risk, evidence, control, assessment, executive, certification, learning, and automation agents.

## Implementation

- Package: `packages/agents`
- Web route: `apps/web/app/agents`
- Admin route: `apps/admin/app/agents`
- Core object: `AgentOperatingSystem`
- Lifecycle: observe, analyze, recommend, execute, validate, learn, report

## Governance

Agent execution plans carry an audit label and mark execution-stage actions as approval-required by default.
