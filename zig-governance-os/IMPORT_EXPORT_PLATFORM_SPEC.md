# Import Export Platform Spec

Imports support enterprise onboarding for projects, frameworks, controls, risks, issues, vendors, evidence, users, assets, policies, tasks, and audits.

Import pipeline: upload, validate, preview, column mapping, transformation, import, verification, and audit.

Exports support audit-ready reporting and portability for operational, compliance, evidence, and executive data sets.

Export pipeline: request, authorization, generation, audit log, download, and archive.

Formats: CSV, Excel, JSON, and future PDF manifests.
