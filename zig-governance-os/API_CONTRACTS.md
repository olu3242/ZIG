# API Contracts

The developer platform uses JWT, service tokens, API keys, RBAC, tenant isolation, rate limiting, request signing, and audit logging.

Initial categories:
- Tenants
- Users
- Projects
- Frameworks
- Controls
- Risks
- Audits
- Evidence
- Automation
- Billing
- Reporting

Every public API must validate tenant context, scope, quota, and audit side effects before mutating data.
