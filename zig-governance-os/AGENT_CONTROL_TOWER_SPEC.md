# Agent Control Tower Specification

The Agent Control Tower is the single pane of glass for inventory, status, health, failures, escalations, certifications, approvals, performance, risk, cost, and utilization.

## Implementation

- Package: `@zig/agent-control-tower`
- Admin route: `/admin/agent-control-tower`
- Persistence: `governed_agents`, scorecards, risk register, approvals, certification, FinOps, telemetry
