# Autonomous Workflow Specification

Autonomous workflows coordinate GRC execution modes across operational processes.

## Implementation

- Package: `packages/autonomous-workflows`
- Engine: `AutonomousWorkflowOrchestrator`
- Workflow types: evidence collection, risk assessment, control testing, vendor review, policy review, audit preparation, certification readiness
- Modes: manual, assisted, autonomous, fully autonomous
- Control: non-manual execution requires approval
